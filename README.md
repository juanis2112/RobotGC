# Encrypted-Control
## Setup
Run the `setup.sh` script to install all library dependencies
```
$ ./setup.sh
```

## Building
To build this project, follow the next steps:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```
## Running 
Terminal 1
```
./bin/system 1 12345
```

Terminal 2
```
./bin/cloud 2 12345
```
